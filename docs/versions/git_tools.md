### gitk
`gitk` 是一个历史记录的图形化查看器。 你可以把它当作是基于 `git log` 和 `git grep` 命令的一个强大的图形操作界面。 当你需要查找过去发生的某次记录，或是可视化查看项目历史的时候，你将会用到这个工具。

### ubuntu合并对比工具
```shell
#安装meld
sudo apt-get install meld
```

编辑`~/.gitconfig`

```shell
[difftool]
    prompt = false
[mergetool]
    prompt = false
[diff]
    tool = meld
[merge]
    tool = meld

```

使用
```shell
git difftool 文件名
git megretool
```
