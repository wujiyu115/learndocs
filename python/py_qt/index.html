开始用pyqt造窗体#

依赖环境#

  • qt5.5.1
  • python3.4
  • pyqt
  • pycharm
  • qt designer

安装#

windows下的安装过程就不再累赘了，讲下mac下的pyqt安装

  • 首先安装sip
#sip安装：
python configure.py
make
make install
  • 编译pyqt
#如果qt安装后没有加入环境变量，指定qmake路径
python configure.py --qmake /Users/far/Qt5.5.1/5.5/clang_64/bin/qmake
make
make install

编译到最后会报错找不到qpeolocation.h，解决办法是从github上下载一个头文件到PyQt-gpl-5.5.1/QtPositioning/qgeolocation.h，地址

工具配置#

  1. pycharm破解 注册时选择在线服务器注册，填入:http://idea.lanyus.com/
  2. 配置pycharm
    • 配置qtdesigner工具,用qtdesigner打开.ui文件
    • 配置pyuic工具,将.ui文件转换成.py文件
    • 然后就可以使用配置的外置工具来编辑.ui文件了
